This section performs the MSiCOR analysis of the MS DMT dataset 
with 3 clusters.

1) Run 'Real_data_analysis_INITIAL_ESTIMATE.m' to get initial 
 estimates of the parameters.

2) Run 'Real_data_analysis_FINAL_ESTIMATE.m' to get final
 estimates of the parameters.

3) Generated 'MS_REAL_DATA_EXAMPLE_SEQ_BETA_num_clus_3.csv' gives
 the estimated coefficient values, noted in Table 7 of the
 MSiCOR paper/draft.

4) Generated 'Cluster_membership_ordered.csv' gives the cluster
 index for each patient.
